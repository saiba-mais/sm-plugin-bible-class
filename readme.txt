The Bible Study plugin includes a button in the wordpress post editor to insert a shortcode that generates a modal
with a bible lesson application. It allows the user to manage the modal style and information via wp-admin.

== Description ==

* Creates an option page in wordpress admin where, is possible to set default general values for the bible study modal, such as 
styles, language and help for the lessons (whatsapp and facebook).

* Creats a button in wordpress post content editor. The button open a modal (described below) for settings and inserts a shortcode which generates
an anchor on the public post which triggers another modal with the lesson.

* Creates a modal triggered by the button (described above), where is possible to to set custom values for the bible study modal instance, such as 
lesson content, styles, language and help for the lessons (whatsapp and facebook).

== Installation ==

1. Upload the entire `bible-study` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find the submenu 'Bible Study Options' menu in your WordPress Settings menu.
